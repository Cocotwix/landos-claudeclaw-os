# Plans Index

Use this file to track the current flow of work.

## How to Use the Plan Folders

- `plans/active/` - work that is currently in progress.
- `plans/pending/` - work that is scoped and ready, but not started.
- `plans/completed/` - finished plans with an execution report.

## Rules

- Keep one source of truth for plan status.
- Move a plan into `completed/` when the work is done.
- Save an execution report for every completed plan.
- Keep plan names descriptive and human-readable.
